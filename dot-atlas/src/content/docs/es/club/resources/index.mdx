## Desarrollo de Software
- **[Documentos de Desarrolladores de Polkadot](https://docs.polkadot.com/):** Tu plataforma de lanzamiento para todo lo relacionado con Polkadot: accede a herramientas, guías y recursos para comenzar rápidamente a construir cadenas personalizadas, desplegar contratos inteligentes y crear dApps.
- **[Polkadot Stack](https://wiki.polkadot.network/docs/build-open-source):** Una visión general de la pila tecnológica de código abierto de Polkadot.
- **[Substrate & Polkadot StackExchange Forum](https://substrate.stackexchange.com/):** Realiza preguntas, encuentra respuestas y colabora en el foro de Stack Overflow para Substrate y Polkadot.
- **[Polkadot Blockchain Academy](https://polkadot.academy/):** Educación blockchain de clase mundial - educando para un mundo más honesto.
- **[Dot Code School](https://dotcodeschool.com/):** Dot Code School es una escuela interactiva en línea que te enseña a construir aplicaciones web3 significativas utilizando el SDK de Polkadot. Aprende a construir tu propia blockchain personalizada desde cero hasta cien.
- **[Polkadot Study](https://polkadot.study/):** Polkadot.study tiene como objetivo ser una plataforma educativa abierta para desarrolladores por desarrolladores, proporcionando contenido educativo de calidad en forma de tutoriales/videos con ejercicios de aprendizaje interactivo y ejercicios de aprendizaje incremental.
- **[Awesome Substrate](https://github.com/polkadot-developers/awesome-substrate):** Una lista curada de proyectos y recursos increíbles relacionados con el marco de desarrollo blockchain Substrate.
- **[Learn with Bruno](https://www.youtube.com/@brunopgalvao/videos):** Canal de YouTube de Bruno Galvao, cofundador de [R0GUE](/club/orgs/software/r0gue), los creadores de POP. Aprende a construir en Polkadot.
- **[Polkadot SDK Best Practices](https://libro.blockdeep.dev/index.html):** Este libro compila ideas esenciales y mejores prácticas derivadas de las revisiones de código exhaustivas de BlockDeep en varios proyectos de Polkadot. El propósito de esta guía es ayudar a desarrolladores, revisores y colaboradores a entender y abordar problemas comunes que pueden surgir en el desarrollo blockchain al construir con el SDK de Polkadot.
- **[Tin Chung Blog](https://blog.chungquantin.com/):** Artículos interesantes sobre el diseño de la red fragmentada de Polkadot y la arquitectura Tanssi.
- **[Polkadot Dev Console](https://dev.papi.how/explorer):** Consola de desarrollador de Polkadot alimentada por polkadot-api.

## Infraestructura
- **[ONE-T - Substrate Blockchain Analytics](https://apps.turboflakes.io/#/dashboard) de TurboFlakes:** Monitorea y explora la red de Polkadot: busca tus Validadores o Pools de Nominación favoritos y visualiza los datos analíticos de blockchain históricos o en tiempo real.
- **[SCOUTY](https://github.com/turboflakes/scouty) de TurboFlakes:** Interfaz de línea de comandos (CLI) para monitorear cadenas basadas en Substrate y conectar elementos.
- **[CRUNCH](https://github.com/turboflakes/crunch) de TurboFlakes:** Interfaz de línea de comandos (CLI) para automatizar fácilmente los pagos de recompensas de staking en cadenas basadas en Substrate.
- **[Guías para Validadores](https://stakeworld.io/docs/validate) de StakeWorld:** Guía de introducción para configurar un nodo validador de Polkadot.
- **[Instantáneas](https://stakeworld.io/docs/snapshot) de Stakeworld:** Herramienta de instantáneas para Polkadot, Kusama y Paseo.
- **[Polkadot Telemetry](https://github.com/paritytech/substrate-telemetry):** Servicio de Telemetría de Polkadot
- **[Polkadot Ansible](https://github.com/Nexus2k/polkadot-ansible):** Herramientas para desplegar un nodo validador de Polkadot/Kusama. Scripts de Ansible para desplegar hasta dos nodos validador de Polkadot basados en Docker. No es necesario compilar ni gestionar dependencias complicadas.
- **[Polkadot Watcher Validator](https://github.com/w3f/polkadot-watcher-validator):** El watcher es una aplicación nodeJs diseñada para conectarse a un nodo basado en Substrate a través de un socket web. Luego puede monitorear el estado del nodo, aprovechando mecanismos como el heartbeat incorporado.
- **[Polkadot Staking Miner](https://github.com/paritytech/polkadot-staking-miner)**
