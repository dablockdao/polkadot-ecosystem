**Subscan est un explorateur de blockchain basé sur Substrate et une plateforme d'analyse** pour ces blockchains. Il permet de rechercher et d'analyser les données de blockchain en équipant les utilisateurs d'une gamme d'outils et de données. Voici ce à quoi les utilisateurs peuvent s'attendre de Subscan :

1. **Explorateur de Blocs** : Un explorateur de blocs permettant aux utilisateurs de voir les détails des blocs, y compris les transactions, les extrinsèques et les événements réalisés dans cette blockchain. Cela inclut la hauteur, le hachage, l'horodatage du bloc, les événements déclenchés et les extrinsèques à l'intérieur, jusqu'à une limite maximale de 1 000 enregistrements.
2. **Détails du Compte** : Subscan peut interroger les détails d'un compte à partir de son adresse, son solde, son historique de transactions et les interactions avec les contrats intelligents ou les fonctionnalités de la gouvernance en chaîne.
3. **Informations sur le Staking** : Subscan fournit des données de staking complètes pour les réseaux disposant d'un mécanisme de staking assurant la sécurité du réseau, y compris des informations sur les validateurs, les nominataires, les récompenses de staking et les pénalités.
4. **Gouvernance et Trésorerie** : Subscan expose les travaux de gouvernance, y compris les propositions, les référendums, les propositions de trésorerie pour les programmes de récompenses. Les utilisateurs peuvent suivre l'état des actions de gouvernance dans leur intégralité, depuis les étapes des propositions, y compris les propositions en cours et terminées, les votes et les résultats de ces votes.
5. **Statistiques du Réseau** : Cela comprend diverses statistiques du réseau, telles que l'offre totale, le volume des transactions, le nombre de comptes actifs, entre autres. Cela aide à déterminer la santé et l'activité du réseau.
6. **Requêtes Personnalisées** : Les requêtes personnalisées permettront aux utilisateurs avancés d'approfondir les données de blockchain de Subscan. Ce cas d'utilisation est plus développé pour les développeurs, chercheurs et analystes qui pourraient avoir besoin de jeux de données spécifiques.
7. **Données Cross-Chain** : En raison de son attention particulière à l'interopérabilité, Subscan fournit également un ensemble d'outils pour explorer les données cross-chain. Les données cross-chain incluent les parachains (dans le cas de Polkadot et Kusama) ainsi que leurs messages et transactions.
8. **Interface Utilisateur** : L'interface utilisateur de Subscan est facile à utiliser pour les personnes techniques et non techniques. Le site dispose de toutes les options de recherche et de navigation sur la plateforme pour un accès facilité à ce que l'utilisateur pourrait vouloir consulter.

Subscan sert d'outil précieux pour toute personne dans l'écosystème Polkadot, **offrant une fenêtre puissante sur les activités en chaîne des blockchains basées sur Substrate**, servant non seulement les développeurs mais aussi les investisseurs, validateurs et chercheurs de ces réseaux.
