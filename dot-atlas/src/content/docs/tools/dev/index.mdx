Polkadot development tools encompass a variety of **frameworks, libraries, and resources that simplify the creation and integration of specialized blockchains within Polkadot**’s secure and scalable multi-chain environment. By leveraging Substrate’s modular architecture, developers can easily customize core functionalities while benefiting from interoperability across the Polkadot network.

|-------------------------------------------|-----------------------------------------------------------------------------------------------|
| [Awesome Polkadot](https://github.com/shawntabrizi/awesome-polkadot?tab=readme-ov-file) | A curated list of awesome resources related to Polkadot by Shawn Tabrizi.                                       |
| [Introduction to Polkadot Cloud](https://forum.polkadot.network/t/the-polkadot-cloud/10670) | Introduction to Polkadot Cloud and how it works.                                               |
| [Rust & Polkadot Workshop](https://www.shawntabrizi.com/rust-and-polkadot-workshop/#/) | Workshop for learning Rust and its integration with Polkadot.                                  |
| [Polkadot Development Documentation](https://docs.polkadot.com/) | Official Polkadot developer documentation.                                                     |
| [Deploy on Paseo Testnet](https://docs.polkadot.com/tutorials/polkadot-sdk/parachains/zero-to-hero/deploy-to-testnet/) | Guide to deploying on the Paseo testnet.                                                       |
| [Build a Custom Pallet](https://docs.polkadot.com/tutorials/polkadot-sdk/parachains/zero-to-hero/build-custom-pallet/) | Tutorial for creating a custom pallet on Polkadot.                                             |
| [Set Up a Parachain Template](https://docs.polkadot.com/tutorials/polkadot-sdk/parachains/zero-to-hero/set-up-a-template) | Instructions for setting up a parachain template.                                              |
| [Polkadot Remix](https://remix.polkadot.io/#lang=en&optimize=false&runs=200&evmVersion=null) | Tool for developing smart contracts in Polkadot.                                               |
| [Polkadot Cloud](https://polkadot.cloud/) | Official website of Polkadot Cloud.                                               |
| [Polkadot Faucet](https://faucet.polkadot.io/) | Service for requesting free test tokens for Polkadot testnets.                                 |
| [Interoperability](https://docs.polkadot.com/develop/toolkit/interoperability/) | Documentation on interoperability within the Polkadot ecosystem.
