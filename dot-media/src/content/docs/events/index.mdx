## Polkadot Decoded
----------------
Polkadot Decoded is a flagship event held annually, bringing together developers, investors, and blockchain enthusiasts. This conference offers in-depth discussions on Polkadot’s latest innovations, network upgrades, and ecosystem growth. It features keynote speeches, panel discussions, and hands-on workshops aimed at fostering collaboration and knowledge sharing within the Polkadot community.

## Sub0
----------------
Sub0 is Polkadot’s dedicated developer conference, designed to connect blockchain developers and enthusiasts. It provides a space for sharing technical knowledge, exploring new tools and technologies, and collaborating on innovative projects. The conference features technical sessions, coding workshops, and networking opportunities, making it an essential event for anyone involved in Web3 development.

## Web3 Summit
----------------
The Web3 Summit is another major event in the Polkadot ecosystem, focused on advancing the decentralized web. This summit attracts top speakers and innovators who discuss the future of Web3, offering a platform for networking and collaborative innovation. The event includes workshops, hackathons, and exhibitions showcasing cutting-edge Web3 technologies.
